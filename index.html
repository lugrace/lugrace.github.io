Bank

Bank is an Android app that helps people save by moving money from a checkings account to a savings account when they are in a blacklisted location. The app implements Capital One's Nessie API and Google Maps APIs. It tracks location, which are stored in a SQLite database. I started it at a hackathon, but am currently working to finish it on my own. It was created with Java and XML.

  • Github: FinalBankApp
  • Date: Nov. 2016 - Present
  • Hackathon: Technica 2016

Conway's Game of Life

This is an interactive simulation of Conway's Game of Life. Users can drag and/or click on clear boxes to set the initial position and then step/run the program to see the cells come to life. It was created using Java.

  • Github: ConwaysGameofLife
  • Date: Spring 2015

Twitter Bot

The program takes in a twitter handle and returns the user's most common word and retweets tweets containing a certain word. I implemented the Twitter API and it was created using Java.

  • Github: TwitterBot
  • Date: Winter 2015

Sudoku Solver

The program can take in a text file with the sudoku puzzle and it prints the solution. It uses recursive backtracking and several heuristics. It was created using Python.

  • Date: Fall 2016

Word Ladder

The program takes in a start word and an end word and outputs the path of words with one letter change to get from the start to the end. It was created using Python and pickling. It can use a depth-first search, breadth-first search, or use the A* algorithm.

  • Date: Fall 2016

Ubuntu Hardening Scripts

I created a script that patches vulnerabilities in the Ubuntu 14.04 operating system. It secures SSH, VSFTP, APACHE2, and other servers. It was created using Bash scripting.
